## 🚀 Tổng Quan Giao Diện

Frontend của PawVerse không chỉ là một trang thương mại điện tử thông thường, mà là một tác phẩm được nhào nặn theo triết lý **"UI-UX PRO MAX"**. Chúng tôi chú trọng vào sự hiện đại, mượt mà và cảm giác "Premium" trong từng điểm chạm.

### 🧬 Triết Lý Thiết Kế
- 🧊 **Glassmorphism**: Tận dụng tối đa các lớp nền kính mờ (backdrop-blur) và viền trắng siêu mảnh để tạo chiều sâu và sự thanh thoát.
- 🎨 **Dynamic Aesthetic**: Hệ thống màu nền được phối hợp giữa Indigo, Slate và các Blob màu động, giúp giao diện luôn tràn đầy sức sống.
- 🖋️ **Elite Typography**: Sử dụng bộ font chữ mạnh mẽ, các tiêu đề được viết hoa toàn bộ (uppercase) với khoảng cách chữ (tracking) rộng - tiêu chuẩn của các dòng thiết bị cao cấp.
- ⚡ **Interactive Motion**: Mỗi hành động từ người dùng đều đi kèm với các hiệu ứng transition và animation (Zoom, Fade-in, Slide) được tinh chỉnh ở mức 700ms - 1000ms để tạo sự sang trọng.

## 🛠️ Công Nghệ Lõi (Frontend Stack)

| Công nghệ | Vai trò |
|-----------|---------|
| **React 19** | Thư viện xây dựng UI hiện đại |
| **Vite 6** | Công cụ build và hot reload siêu tốc |
| **TailwindCSS 3.4** | Framework CSS utility-first cho tinh chỉnh giao diện |
| **Zustand 5** | Quản lý trạng thái ứng dụng (Auth, Cart, Navigation) |
| **TanStack Query 5** | Quản lý cache, đồng bộ dữ liệu Server-side |
| **Lucide React** | Bộ icon vector sắc nét và đồng nhất |
| **Framer Motion** | Xử lý các chuyển động phức tạp |

## 📂 Các Phân Khu Chức Năng Chính

### 📑 1. Authentication Flow (Elite Design)
*   **Login/Register**: Giao diện tối giản với hệ thống Decorative Icons linh hoạt.
*   **Forgot Password**: Luồng đặt lại mật khẩu đa bước (Multi-step) trực quan.
*   **OAuth2 Integrations**: Google, Github, Discord.

### 📊 2. Admin Dashboard (Elite OS Style)
*   **Quản lý người dùng**: Giao diện bảng (Table) cực kỳ hiện đại với hệ thống Badge và nút thao tác tinh gọn.
*   **User Detail Modal**: Thiết kế theo phong cách "Trung tâm bảo mật" với Metadata chi tiết và bố cục logic.
*   **Analytics**: Biểu đồ trực quan và Statistics card thông minh.

### 🛒 3. E-commerce & Services
*   **Cart System**: Tối ưu hóa kích thước nội dung, tạo cảm giác tinh tế và gọn gàng.
*   **Product Gallery**: Trình diễn sản phẩm với hiệu ứng Atmospheric Bleed cho hình ảnh.
*   **Pet Management**: Hệ thống quản lý hồ sơ thú cưng cá nhân hóa.

## ⚙️ Hướng Dẫn Cài Đặt (Cho Developer)

1. **Di chuyển vào thư mục frontend:**
   ```bash
   cd frontend
   ```

2. **Cài đặt các thư viện cần thiết:**
   ```bash
   npm install
   ```

3. **Khởi chạy môi trường phát triển:**
   ```bash
   npm run dev
   ```

4. **Truy cập:**
   Mở trình duyệt tại địa chỉ `http://localhost:5173`.
